# Agentic Development Framework

Agentic Development Framework is a contract-driven meta-framework for coordinating specialized agents that execute software milestones through explicit phases, gates, review, and git lifecycle controls.

The framework is intentionally agnostic to any single product, stack, or domain. Factory Build Agent (FBA) is expected to become a future adapter or use case, not the core of this repository.

Current status: extraction/bootstrap. The repository contains only the initial package shell and coordination contract. Agents, schemas, runtime components, utilities, and integrations will be introduced later under explicit policy.
